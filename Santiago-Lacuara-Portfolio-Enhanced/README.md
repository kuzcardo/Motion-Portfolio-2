# Santiago Lacuara — Motion Graphics Portfolio

Portfolio personal de **Santiago Lacuara**, Motion Graphics Designer.
Sitio de una sola página con animaciones, modo bilingüe (EN/ES) y galería de reels por estilo de diseño.

## ✨ Características

- Hero animado con título tipo máquina de escribir y halo interactivo que sigue el cursor.
- Grid de fondo global con efecto parallax y manchas de color en movimiento.
- Galería de **7 estilos de diseño**, cada uno con sus reels de YouTube en un modal.
- Marquee de estilos, sección "Sobre mí" y contacto.
- Bilingüe Español / Inglés.
- Tipografía Syne + Space Grotesk, paleta oscura con acento ámbar.

## 🛠️ Stack

- [React 19](https://react.dev/)
- [Vite 6](https://vitejs.dev/)
- [Tailwind CSS 4](https://tailwindcss.com/)
- [Motion](https://motion.dev/) (animaciones)
- TypeScript

## 🚀 Desarrollo

```bash
npm install
npm run dev
```

Abre el enlace que aparece en consola (por defecto `http://localhost:3000`).

## 📦 Build de producción

```bash
npm run build
npm run preview
```

## 📁 Estructura

```
src/
  components/   Navbar, HeroAboutSection, ProjectGrid, Marquee...
  context/      LanguageContext (EN/ES), ThemeContext
  index.css     Sistema de estilos (tokens, animaciones)
public/         Imágenes de las carátulas
```
